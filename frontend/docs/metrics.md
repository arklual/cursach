# Метрики и формулы

| Метрика | Формула | Пояснение |
| --- | --- | --- |
| `N_reached` | \(N = \lvert\{user\_id\_hash\}\rvert\) с `node_reached` | Сколько уникальных пользователей достигло ноду |
| `k_converted` | Кол-во событий `node_action` со статусом success | Число успешных действий |
| `\hat p` | \(\hat p = k / N\) | Выборочная доля конверсий |
| `Var(\hat p)` | \(Var = \hat p(1-\hat p)/N\) | Дисперсия оценки Бернулли |
| `CI95%` | \(\hat p \pm z_{0.975}\sqrt{Var}\) | Доверительный интервал 95% |
| `Δ` | \(p_B - p_A\) | Разница между вариантами |
| `p\text{-value}` | \(2(1-\Phi(|z|))\) | Вероятность наблюдать Δ при H₀ |

**Power/sample калькулятор:** \(n \approx \frac{(z_{1-\alpha/2}+z_{1-\beta})^2 p(1-p)}{d^2}\)

где \(p\) — ожидаемая базовая конверсия, \(d\) — минимально значимая дельта, \(z\) — квантиль нормального распределения.
