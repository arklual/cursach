
| **Технология/Инструмент** | **Описание**                                                                  | **Причины выбора**                                                                                                                                                                                                                                                                                                                                                 |
| ------------------------- | ----------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Java**                  | Основной язык программирования для разработки серверной части (Backend).      | • **Строгая типизация:** Обеспечивает надежность кода и упрощает поддержку сложной логики работы с графами.<br><br>• **Многопоточность:** Эффективная работа с параллельными процессами и фоновыми задачами (в отличие от однопоточных аналогов).<br><br>• **Экосистема:** Огромный выбор готовых библиотек и инструментов для решения любых интеграционных задач. |
| **Spring (Spring Boot)**  | Фреймворк для создания архитектуры приложения и API.                          | • **Скорость разработки:** Готовые модули для работы с БД, безопасностью и веб-запросами ускоряют создание MVP.<br><br>• **Управление зависимостями:** Упрощает конфигурацию компонентов и поддержку чистоты архитектуры.<br><br>• **Стандарт индустрии:** Легкость в поиске решений, документации и специалистов.                                                 |
| **PostgreSQL**            | Реляционная база данных для хранения данных и состояния системы.              | • **Гибкость данных:** Отличная поддержка JSON позволяет хранить структуры воркфлоу и неструктурированные данные в одной БД.<br><br>• **Надежность (ACID):** Гарантирует сохранность данных и целостность транзакций при сбоях.<br><br>• **Функциональность:** Мощные возможности индексации и масштабирования "из коробки".                                       |
| **Angular**               | Фреймворк для разработки клиентской части (Frontend) и визуального редактора. | • **Структурированность:** Четкая архитектура навязывает порядок в коде, что критично для создания сложных интерфейсов.<br><br>• **TypeScript:** Использование типов на клиенте синхронизируется с подходом Java на бэкенде.<br><br>• **Производительность:** Оптимизирован для работы с большими и динамичными веб-приложениями.                                  |
