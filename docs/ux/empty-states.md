**Сейчас:**
- "Simulate 500 users" — неясно, зачем это нужно

### 2.3. Пояснения статистических терминов

**Вместо:** `p̂ = 0.25`

**Новый формат:**
```
Конверсия (p̂): 25%
├─ Что это: Доля пользователей, выполнивших целевое действие
├─ Формула: k / N = 125 / 500 = 0.25
└─ Интерпретация: Каждый 4-й пользователь совершил покупку
```

**Вместо:** `CI95% = [0.21, 0.29]`

**Новый формат:**
```
Доверительный интервал: 21% – 29%
├─ Что это: Диапазон, где с вероятностью 95% находится истинная конверсия
├─ Почему важно: Показывает точность оценки
└─ Интерпретация: Мы на 95% уверены, что реальная конверсия между 21% и 29%
```

**Вместо:** `p-value = 0.043`

**Новый формат:**
```
Значимость (p-value): 0.043
├─ Что это: Вероятность, что разница случайна
├─ Порог: < 0.05 считается значимым
└─ Вывод: ✅ Разница реальна (p < 0.05)
```
