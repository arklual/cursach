#### StatisticsTermsService
**Файл:** `frontend/src/app/services/statistics-terms.service.ts`

**Функционал:**
- ✅ Словарь статистических терминов (8 терминов)
- ✅ 2 режима: Базовый ("Конверсия") / Эксперт ("p̂")
- ✅ Пояснения, формулы, интерпретации
- ✅ Методы: `getTerm()`, `getTooltip()`, `getRecommendation()`

**Термины:**
| Ключ | Базовый | Эксперт |
|------|---------|---------|
| conversion | Конверсия | p̂ (sample proportion) |
| ci95 | Доверительный интервал | CI95% |
| pValue | Значимость | p-value |
| variance | Разброс | Var(p̂) |
| delta | Разница | Δ |
| power | Мощность | Statistical Power |
| sampleSize | Размер выборки | N |
| trafficAllocation | Распределение трафика | Traffic Allocation |
