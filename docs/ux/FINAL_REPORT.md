### 🎨 UI/UX Особенности

#### Цветовая схема статусов
```
⚪ Pending   —  #94a3b8 (Slate)
🟡 Running   —  #f59e0b (Amber)  — пульсация
🟢 Success   —  #22c55e (Green)  — зелёная рамка
🔴 Error     —  #ef4444 (Red)    — красная рамка
```

#### Анимации
```css
/* Пульсация ноды */
@keyframes node-pulse {
  0%, 100% { box-shadow: 0 0 0 0 rgba(245, 158, 11, 0.4); }
  50% { box-shadow: 0 0 0 12px rgba(245, 158, 11, 0); }
}

/* Вращение спиннера */
@keyframes spin {
  to { transform: rotate(360deg); }
}

/* Поток данных по ребру */
@keyframes edge-flow {
  0% { stroke-dashoffset: 20; }
  100% { stroke-dashoffset: -20; }
}
```

#### Визуальные эффекты
- **Нода выполняется:** Жёлтая пульсирующая рамка + иконка ⟳
- **Нода успешна:** Зелёная рамка + иконка ✓ + время выполнения
- **Нода с ошибкой:** Красная рамка + иконка ✕
- **Ребро активно:** Пунктирная линия "бежит" по ребру

### 🚀 Как использовать

#### Запуск исполнения
```typescript
// Кнопка "Симуляция (500)" в header
simulateRun(500);
```

#### Наблюдение за прогрессом
1. Ноды подсвечиваются по мере выполнения
2. Рёбра анимируются при передаче данных
3. Progress bar показывает общий прогресс
4. Header показывает "⟳ 2/5" (выполняется 2 из 5)

#### Просмотр деталей
1. Клик на ноду в ExecutionPanel
2. Input/output отображаются в JSON формате
3. Кнопка Copy для копирования
4. Время выполнения в миллисекундах
